# Toolboxer

## 核心功能

### 1. 目录分析
- 树形结构展示目录内容
- 支持文件大小、权限、修改时间显示
- 支持深度控制与模式过滤

### 2. 端口查看
- 显示端口占用进程信息
- 支持显示所有TCP/UDP端口
- 支持过滤监听状态端口


## 使用示例

```bash
# 快速查看当前目录结构（2层深度）
toolboxer tree -d 2

# 端口占用
toolboxer portown -l

```

## 参数说明

### tree 命令
| 参数 | 简写 | 说明 |
|------|------|-----|
| --depth | -d | 遍历深度限制 |
| --all | -a | 显示隐藏文件 |
| --sort-size | -s | 按文件大小排序 |
| --human-size | -S | 人性化显示大小 |
| --filter | -f | 文件名模式过滤 |
